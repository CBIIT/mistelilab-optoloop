# mistelilab-optoloop

### General Information

This is a repo containing the R code used to generate the results included in the [Stortz et. al, Cell Science 2026](https://journals.biologists.com/jcs/article/doi/10.1242/jcs.264574/370392/OptoLoop-An-optogenetic-tool-to-probe-the) publication from the Misteli Lab at the National Cancer Institute/NIH. The manuscript introduces a new experimental tool, called OptoLoop, which can be used to manipulate 3D genome structure in live cells in a rapid fashion.

To rerun the R code, first download the appropriate input data from the associated [Figshare repo](https://figshare.com/projects/mistelilab_optoloop/267052). The `code_data_sources_key_table.csv` file contains the information on where to find the image analysis results data used as the input for the statistical analysis and plots generation in each figure of the paper.

**Title:** OptoLoop: An optogenic tool to probe the functional role of genome organization.

**Authors:** Martin Stortz, Adib Keikhosravi, Gianluca Pegoraro, Tom Misteli.

**DOI:** 10.1242/jcs.264574
